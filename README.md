# Blockchain-Based Pharmacy & Prescription Management System

A secure pharmacy management platform powered by blockchain technology that connects pharmacists and patients.  
The system ensures prescriptions are tamper-proof, traceable, and accessible only by authorized users.

---

## Problem Statement

Traditional prescription systems are often paper-based or stored in centralized databases.  
This creates risks such as:

- Prescription tampering or forgery  
- Loss of medical records  
- Lack of transparency between pharmacies and patients  
- Difficulty tracking medicine history  

---

## Solution

This platform uses blockchain technology to securely store and manage prescription data.

- Pharmacists create verified prescriptions  
- Patients receive prescriptions via QR code or ID  
- Records cannot be altered by patients  
- Only authorized pharmacists can update prescriptions  
- Medicine history becomes transparent and traceable  

---

## Features

### Pharmacist Portal
- Register pharmacy account using license verification  
- Create digital prescriptions with:
  - Medicine name
  - Dosage
  - Instructions
  - Time schedule
- Update or modify prescriptions securely  
- Scan patient QR to access records  
- Search patients using unique ID  

### Patient Portal
- Create consumer account  
- Receive prescriptions via QR or ID  
- View medicine details and instructions  
- Cannot edit or modify prescription data  

### Blockchain Security
- Immutable prescription records  
- Tamper-proof data storage  
- Transparent medicine history tracking  
- Secure access control for pharmacists
